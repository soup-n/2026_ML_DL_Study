## 스터디 규칙
- 과제 미제출 또는 무단 결석 1회시 트랙 경고 1회 부여
- 챌린저 트랙과 병행 중인 경우, 2회시 트랙 경고 1회 부여
  	- 단, 결석 사유가 합당한 경우(장례식/결혼식), 결석 사유서 제출 후 대체 과제 제출 시 결석 인정
	- 결석 사유가 합당하지 않은 경우(여행), 스터디 3일 전까지 제출해야 심사 대상 포함
	- 과제는 제출 마감일 기준 3일이 경과하면 미제출로 간주
- 과제 지각 또는 수업 지각 2회 누적 시 트랙 경고 1회
- **과제 지각은 해당 수업 시작(1시) 전까지, 수업 지각은 수업 시작 후 15분 전까지**
